# make-string
## String 클래스를 C/C++/Java 로 구성해보는 연습 프로젝트

### 1. C lang
- C 포인터/구조체를 활용한 string 구현
- strlen/strcpy/strcat/strrchr/atoi/strtol 등 string 내부 함수 구현

### 학습한 내용
- Java의 클래스를 완벽히 구현할 수는 없지만, 구조체를 활용해 모방해볼 수 있음
- String 구조체의 포인터를 반환
- Java new 의 경우, 지정된 타입에 해당하는 메모리를 반환하게 된다.
- 스코프를 벗어나면 스택 상 메모리는 모두 해제된다
- 문자열 리터럴은 ROM에 저장됨(읽기 전용 영역) -> char [], char * 모두 동일한 배열 형태지만 char *는 수정 안된다.
